# Tweet 写作规则（中英文）

> 语气 baseline 来自 Diff 的 polisher skill。这里只列 tweet-specific 规则。

## 通用

- **一条 tweet = 一个观点**。两个观点 → 拆成 thread。
- **前两行钩住**。X 的"show more"折叠通常出现在 ~150 字符；最重要的话放最前。
- **数字优于形容词**。"36 天" > "最近"，"3 小时" > "好久"。
- **具体优于抽象**。"贝塔咖啡" > "一家咖啡馆"。
- **不用 hashtag**（Diff 从不用）。
- **emoji 限制**：🧵（thread 标记）、📋（copy 提示）、其他单条 ≤1 个，且必须功能性。

## 中文规则

### 句式
- 短句优先。能用句号别用逗号。
- 一段最多 3 行，超过就空行分段。
- 数字 / 英文用半角，前后空格："36 天"、"$0"、"AI 时代"。

### 词汇
- 保留 Diff 的招牌词：`我想` / `我觉得` / `其实` / `但是` / `说实话` / `所幸` / `坦白说`
- 保留口语化的"啊 / 呢 / 吧"——只要 polisher 没删
- **禁用清单**（公司腔/营销腔）：
  - 干货、收割、闭环、方法论、赋能、抓手、对齐、深耕、复盘（动词除外）
  - 颠覆、风口、红利、爆款、出圈
  - "致敬"、"献给"开头
  - 万能金句结尾："共勉"、"加油"、"愿你..."

### 标点
- 中文标点：，。：；「」（不用《》除非书名/作品名）
- 引号优先用「」而非 ""
- 不用句末感叹号串（!!! → 至多一个 ！，多数情况下省掉）

### 长度
- 单条目标：80–280 中文字符
- Premium 上限：可达 600，但超 400 几乎没人读到底
- 钩子（首行）：≤30 字最佳

## 英文规则

### 体裁
- 第一人称 indie hacker / design Twitter 口语
- 短句 + 偶尔长破折号 + 偶尔 (parens)
- show, don't tell：用具体场景代替"Here's why X matters"

### 禁用清单
- "Here's why..." / "The truth is..." / "Hot take:" 开头（这是 AI 味重灾区）
- "🧵 a thread on..." 直接进 thread 即可
- "Game-changer" / "10x" / "this changed everything"
- "Let that sink in"
- 任何 "💡 thought of the day"

### 长度
- 单条目标：180–260 字符（X 算法这区间最甜）
- 不要逼近 280 字符上限——给 retweet 加评论留空间

### 词法
- 用 contractions（it's / you're / don't）
- 数字用阿拉伯数字而非英文（"3 hours" not "three hours"）
- 时间格式："36 days ago"、"2am"（不用 12-hour 大写）

## Thread 规则

### 长度
- 最少 5 条，最多 15 条
- 中文 thread 每条 200–280 字符
- 英文 thread 每条 180–260 字符
- 严禁"为凑数而拆"——能 5 条说完不要拖到 10 条

### 结构
- **Tweet 1**（hook）：今天发个故事/数据/反直觉判断 + "下面 🧵"
- **Tweet 2**（setup）：背景、起因
- **Tweet 3 至 N-2**（beats）：故事或论证的中段
- **Tweet N-1**（lessons）：3 条以内 takeaway，编号
- **Tweet N**（CTA）：软引导 follow + 完整版链接（如有）

### Thread 内首条字符数
- 控制在 220–270，给"显示更多"留余地，让算法判定为有价值长内容

## 时区与排程

- 所有 scheduled_at 用 `-05:00`（Dallas Central Time）
- 中文主贴：07:30 CT = 北京 20:30（中文圈黄金）
- 英文主贴：12:00 CT（美东 13:00，indie 圈活跃）
- 晚贴：21:00 CT
- 长 thread：周日 10:00 CT（美国周日上午 = 周一中文晨刷）

## 跨语言隔离

- 中英文 tweet 在 `tweets.js` 里通过 `lang` 字段区分
- 不要做"中英对照"输出
- 同一观点可以中英都产，但**写法、例子、钩子都应该不同**
- 中文里出现的本土文化梗（贝塔咖啡、文二西路、徐州）→ 英文版要么换具体例子要么完全换 angle

## 必须 mark needs_review = true 的情况

- 来源是 notes/Drafts/（草稿）
- 涉及具体姓名（除已公开人物如王翌）
- 长 thread（≥10 条）
- 涉及任何金额、薪资、收入数字
- 引用了别人原话
- 和最近 7 天某条 tweet 主题相似度 >70%
